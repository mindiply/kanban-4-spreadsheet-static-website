User guide

Kanban your spreadsheet lets you change field values graphically.

The application does not operate directly on the files you imported. The changes you make have to be re-exported to a csv or Excel file.

Please, do not overwrite your original file, especially if it is an Excel file, as we are unable to re-export your formatting and formulas at this time.

The suggested process to re-import the data after you modified it using our tool is:

1. Import the data into Kanban your spreadsheet.

2. Export the data to a new spreadsheet after changing it in Kanban your spreadsheet.

3. Copy the records in the exported spreadsheet.

4. Paste the records in a copy of your original spreadsheet.

We plan for a more efficient process in the future. Please let us know what you would like the process to be!

The landing page

It all starts with an import. Select a csv or Excel file, and click on the Open button. Should nothing happen, we are unable to find data to process. Please let us know; we may be able to fix it.

Setting up the data source

The data setup page is where you define which fields you want to change their values, and optionally, which fields make up the label for the cards that correspond to the records.

When you tick the checkbox in the row Can be used in kanban, that field will be one you can choose to assign values to in the Matrix and Kanban screens. When we import your data, we exclude fields with too many distinct values.

When you tick the checkbox in the row Used for label field, we will use that field to generate the label for the cards representing the records in the spreadsheet you imported. You can deselect all the fields, and the order in which you select them will determine the label's structure.

We only show a few of the records you imported to demonstrate how we understand your data and make it easier for you to choose the fields you want to work with.

Once you are happy with your setting, go to either the Kanban or Matrix view using the buttons at the bottom of the screen.

Don't worry too much about getting everything right; you can always come back to this screen to change the settings.

You can also import new data from this screen using the Add Excel or CSV file.

If you imported multiple data sources, or if you imported an Excel file that has multiple sheets with data, you can switch to a specific table of records using the Select data source drop-down button at the top-left of the screen.

The Kanban screen

On the Kanban page, you can change the value of a field at a time. Each value is represented by a column, with its value at the top. Within the column, each card represents a record in your spreadsheet. You can change the value of that field by moving the cards from one column to the other.

If you don't like the order of the values, you can drag and drop the column to the left or right.

After you finish assigning values to one field, you can use the Field to filter by drop-down to switch to a different field.

You can save your changes by clicking on either the Download Excel Workbook or Download CSV button.

The small button switch in the top-right corner lets you quickly toggle between the matrix and kanban screens.

If you click on the Data source setting link in the bottom-left corner of the screen, you can change the data source settings.

The matrix Screen

The matrix page creates a matrix with the values of two fields in your spreadsheet.

If you have urgency and importance, for instance, you've just got yourself an Eisenhower matrix.

This page works similarly to the Kanban page, but when you move a card between cells, you are changing two field values at the same time.

The Row field and Column field drop-down buttons allow you to change what field is used for each axis.

FAQ

When would I use this tool?

You use spreadsheets to manage a list of items, but you'd prefer a graphical interface to assign some of the fields. If, for instance, you have a product backlog, you may like to assign a KANO type by drag-and-drop, and also get an idea of the distribution of records across the potential values.

What about the cookie notice?

This app runs only on your browser; there is no back-end server that it communicates with. We don't collect any information at all. Still, feel free to get in touch with us!
